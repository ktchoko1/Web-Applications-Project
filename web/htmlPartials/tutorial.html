Filter Table Framework Tutorial with Java Script

The tutorial shows how to create filter table framework style using java script. The java script function "FilterTable()" that filter the table is a public function and contains four privates functions and one method. Follow these step to create the framework: create a function and give it a name like FilterTable().

Step5:

Create the function the filter the table. It takes the input from the search bar and loop through the row and filter the table according to the letter characters of the word enter. 
   
        function dataFilter(inputBar) {

            var input, filter, table, tr, td, i;
            input = document.getElementById(inputBar);
            filter = input.value.toUpperCase();
            table = filterObj;
            tr = table.getElementsByTagName("tr");

            for (i = 0; i < tr.length; i++) {
                td = tr[i].getElementsByTagName("td")[0];
                if (td) {
                    if (td.innerHTML.toUpperCase().indexOf(filter) > -1) {
                        tr[i].style.display = "";
                    } else {
                        tr[i].style.display = "none";
                    }
                }
            }
        }
